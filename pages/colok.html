Colok

Le projet

Dans le cadre de notre premier projet de groupe chez Interface 3 nous avons été amenés à développer un site web par groupe de 3.

Outils utilisés

  • HTML
  • SCSS
  • PHP
  • Tailwind
  • Git
  • GitHub

Les tâches

Par groupes de trois, chacun a endossé un rôle. Les différentes tâches auxquelles j'ai participé sont les idées qui constituent le projet, la rédaction du contenu, le design, le prototype et le développement du site web.

Compétences développées

  • HTML
  • SASS
  • Collaboration en équipe
  • Test
  • Résolution de problèmes
  • Créativité
  • Manipulation de Git et GitHub

Colok

Colok est une plateforme web conçue pour faciliter la vie quotidienne des colocataires. Elle offre la possibilité d'organiser facilement les tâches ménagères, de planifier des événements via un agenda intégré, et de partager des idées grâce à notre espace Boîte à idées.
