Agrégateur de contenu

Le projet

Dans le cadre de mon projet Node.js chez Interface 3, j'ai eu l'opportunité de concevoir un agrégateur de contenu. Ce dernier est conçu pour afficher des éléments provenant d'autres sites web en exploitant des API et des flux RSS.

Outils utilisés

  • HTML
  • CSS
  • JavaScript
  • NodeJS
  • Flux RSS
  • API

Les tâches

  • Importer des modules :
    • https pour effectuer des requêtes HTTPS.
    • xml2js pour parser les réponses XML.
    • express pour la gestion du serveur web.
    • Le module ejs pour la génération de pages web dynamiques avec des templates EJS.
  • Configurer le serveur Express
  • Écouter le serveur sur le port 8080
  • Définir les routes
  • Initialiser les données à afficher
  • Stocker les données en question
  • Afficher les données récupérées sur une page HTML
  • Styliser la page à l’aide de CSS

Compétences développées

Ce projet m'a permis d'explorer et de mettre en œuvre des fonctionnalités avancées liées à la récupération et à l'affichage dynamique de données à partir de différentes sources en ligne.

Agrégateur de contenu

C'est un agrégateur de contenu conçu pour offrir à l'utilisateur la possibilité de consulter quotidiennement la météo, des recettes de cuisine, des informations liées au web, des citations, et éventuellement des formations provenant de Dorifor. Découvrez l'écran du projet.
